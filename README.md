# Task-1--Web-Scrapping

## Objective
The objective of this project is to scrape book information from the "Books to Scrape" website using Python and save the extracted data into a CSV file.

## Website
https://books.toscrape.com/

## Tools & Libraries Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook

## Data Extracted
- Book Title
- Price
- Rating

## Project Files
- `book_scraping.ipynb` – Jupyter Notebook containing the scraping code.
- `books.csv` – CSV file with the extracted book data.
- `README.md` – Project documentation.

## Workflow
1. Sent an HTTP request to the website.
2. Parsed the HTML using BeautifulSoup.
3. Extracted book title, price, and rating.
4. Stored the data in a Pandas DataFrame.
5. Exported the data to `books.csv`.

## Output
A structured CSV dataset containing book details that can be used for further data analysis.
